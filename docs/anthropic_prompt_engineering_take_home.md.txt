# Anthropic Prompt Engineering Take-Home Assignment

---

## Context

As AI capabilities advance rapidly, engineers at Anthropic face a unique challenge:
building tools and experiences that are genuinely useful, trustworthy, and delightful.

The best products:
- Demonstrate taste
- Handle complexity gracefully
- Solve real problems in non-obvious ways

This assignment evaluates your prompt engineering and evaluation skills through a focused, practical task.

Include time spent in your written rationale.

---

## The Task

Build a system that uses **Claude + Wikipedia** to answer questions and evaluate its performance.

### Requirements

1. **System Prompt + Tool Design**
   - Design a system prompt for an LLM with access to:
     ```
     search_wikipedia(query: str)
     ```
   - The prompt should guide effective use of Wikipedia for high-quality answers

2. **Wikipedia Retrieval Integration**
   - Implement a working end-to-end system
   - Allowed approaches:
     - MediaWiki API
     - Wikipedia dump
     - Local search index
     - Any custom method

3. **Evaluation Suite**
   - Build a system to evaluate performance across test cases
